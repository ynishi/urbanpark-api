# urbanpark-api
* Provide api for Japanese urban park planning data
* using https://github.com/ynishi/urbanparkjp
* using lambda-sdk-go

## Usage
* clone
```
git clone git@github.com:ynishi/urbanpark-api.git
```
* build
```
make
```
* setup apigateway and lambda with handler.zip
